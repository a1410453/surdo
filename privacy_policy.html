2. Основные понятия

  • 2.1. Автоматизированная обработка персональных данных – обработка персональных данных с использованием вычислительной техники.
  • 2.2. Блокирование персональных данных – временная приостановка обработки персональных данных (за исключением случаев, когда обработка необходима для уточнения персональных данных).
  • 2.3. Мобильное приложение – программное средство для мобильных устройств, обеспечивающее их функциональность и взаимодействие с операционной системой.
  • 2.4. Информационная система персональных данных — совокупность данных о пользователях, обрабатываемых в рамках функционирования мобильного приложения.
  • 2.5. Обезличивание персональных данных — процесс, в результате которого становится невозможным определить принадлежность персональных данных конкретному пользователю.
  • 2.6. Обработка персональных данных – любые операции с данными, включая сбор, запись, систематизацию, хранение, обновление, изменение, использование, передачу, обезличивание, блокирование, удаление, уничтожение.
  • 2.7. Оператор – лицо или организация, осуществляющая обработку персональных данных.
  • 2.8. Персональные данные – информация, относящаяся к определенному пользователю мобильного приложения.
  • 2.9. Пользователь – любой человек, использующий мобильное приложение.
  • 2.10. Предоставление персональных данных – раскрытие информации определенному лицу или кругу лиц.
  • 2.11. Распространение персональных данных – передача данных неопределенному кругу лиц или их ознакомление с персональными данными.
  • 2.12. Трансграничная передача персональных данных – передача данных за пределы территории государства.
  • 2.13. Уничтожение персональных данных – действия, приводящие к необратимому уничтожению данных в информационной системе.
